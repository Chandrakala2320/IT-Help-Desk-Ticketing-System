# Automated IT Help Desk Ticket Management System

This project implements an automated Incident Management workflow using ServiceNow ITSM. It focuses on reducing manual effort in handling IT support tickets by automating classification, assignment, SLA tracking, and escalation.

## Platform and Tools
- ServiceNow ITSM
- Workflow Designer
- Business Rules
- Script Activities
- SLA Management
- Notifications and Reports

## Project Overview
The workflow automatically processes new incidents created via email or portal, categorizes them based on keywords, assigns them to the appropriate support team, and monitors SLA compliance with auto-escalation for delayed tickets.
